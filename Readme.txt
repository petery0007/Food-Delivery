## 一、系统简介

### 1.1 系统概述
绿色有机食品配送系统是一个基于Web的在线食品购物平台，专注于提供绿色有机食品的展示、购买和配送服务。系统采用前后端分离架构，为用户提供便捷的在线购物体验，为管理员提供完善的商品和订单管理能力，为配送员提供高效的配送管理工具。

### 1.2 系统特色
- 🌱 **专注绿色食品**：提供新鲜蔬菜、有机蔬菜、新鲜水果等健康食品
- 🔒 **安全可靠**：JWT身份认证，角色权限隔离，数据安全保障
- 💳 **多种支付方式**：支持余额支付和货到付款两种方式
- 📱 **界面友好**：响应式设计，适配主流浏览器
- ⚡ **高效配送**：智能配送员分配，实时订单状态跟踪

### 1.3 适用角色
- **普通用户**：浏览商品、下单购物、查看订单、评价商品
- **管理员**：商品管理、订单管理、用户管理、配送员管理
- **配送员**：查看配送订单、更新配送状态

## 二、系统要求

### 2.1 运行环境要求

#### 客户端（用户端）
- **浏览器**：Chrome 120+ 或 Edge 120+（推荐最新版本）
- **分辨率**：支持1920×1080、1366×768、1440×900等主流分辨率
- **网络**：稳定的互联网连接

#### 服务器端（部署用）
- **操作系统**：Windows/Linux/Mac OS
- **JDK版本**：JDK 17或更高版本
- **Node.js**：Node.js 18或更高版本
- **数据库**：MySQL 8.0
- **构建工具**：Maven 3.6+

### 2.2 性能指标
- 页面加载时间：≤ 2秒
- 接口响应时间：≤ 3秒
- 登录响应时间：≤ 1秒
- 支持并发用户数：≥ 50人

## 三、安装部署

### 3.1 数据库配置

#### 步骤1：创建数据库
sql
CREATE DATABASE food_delivery DEFAULT CHARACTER
SET utf8mb4 COLLATE utf8mb4_unicode_ci;
#### 步骤2：导入数据表
按以下顺序执行SQL文件：
bash
mysql -u root -p food_delivery < sql/user.sql
mysql -u root -p food_delivery < sql/products.sql
 mysql -u root -p food_delivery < sql/cart.sql
 mysql -u root -p food_delivery < sql/orders.sql
 mysql -u root -p food_delivery < sql/comment.sql
 ### 3.2 后端部署

 #### 步骤1：修改配置文件
 编辑 `src/main/resources/application.yml`：
 yaml spring:
 datasource:
 url:
 jdbc:mysql://localhost:3306/food_delivery?useSSL=false&serverTimezone=Asia/Shanghai
 username: root
 password: your_password # 修改为您的数据库密码（其他部分不变）
 #### 步骤2：编译打包
bash
mvn clean package
#### 步骤3：启动后端服务
bash
java -jar target/Food-Delivery-0.0.1-SNAPSHOT.jar
后端服务将在 **http://localhost:8081** 启动

### 3.3 前端部署

#### 步骤1：进入前端目录
bash
 cd green-food-web
 #### 步骤2：安装依赖
bash
npm install
#### 步骤3：启动前端服务（开发模式）
bash
 npm run serve
 前端服务将在 **http://localhost:8080** 启动

 #### 步骤4：生产环境打包（可选）
bash
npm run build
### 3.4 访问系统
在浏览器中访问：**http://localhost:8080**

## 四、用户操作指南

### 4.1 注册账号

1. 打开系统首页，点击"注册"按钮
2. 填写注册信息：
   - **用户名**：4-20个字符，只能包含字母、数字、下划线
   - **密码**：6-20个字符，建议包含字母和数字
   - **确认密码**：必须与密码一致
   - **手机号**：11位有效手机号码
3. 点击"注册"按钮
4. 注册成功后自动跳转到登录页面

**注意事项：**
- 用户名和手机号必须唯一，不能重复注册
- 请牢记您的登录密码
- 系统会自动将新注册用户设置为"普通用户"角色

### 4.2 登录系统

1. 在登录页面输入用户名和密码
2. 选择角色：**普通用户**
3. 点击"登录"按钮
4. 登录成功后进入商品列表页面

**安全提示：**
- 连续输错密码5次将锁定账号30分钟
- 登录状态有效期为24小时
- 请勿在公共电脑上保存登录信息

### 4.3 浏览商品

#### 查看商品列表
- 登录后默认显示所有上架商品
- 商品以卡片形式展示，包含：
  - 商品图片
  - 商品名称
  - 产地信息
  - 商品简介
  - 规格分类
  - 价格
  - 库存状态

#### 搜索商品
1. 在搜索框中输入关键词（商品名称或产地）
2. 系统实时显示匹配结果
3. 点击商品卡片查看详情

#### 筛选商品
1. 在规格筛选下拉框中选择分类：
   - 新鲜蔬菜
   - 有机蔬菜
   - 新鲜水果
2. 系统自动过滤显示对应分类的商品

#### 分页浏览
- 每页默认显示12个商品
- 可切换每页显示数量（12/24/36）
- 点击页码或"上一页"/"下一页"翻页

### 4.4 查看商品详情

1. 点击商品卡片进入详情页
2. 查看完整商品信息：
   - 高清商品图片
   - 详细商品描述
   - 产地信息
   - 规格参数
   - 价格和库存
   - 历史评价
3. 可选择：
   - **加入购物车**：添加到购物车继续购物
   - **立即购买**：直接下单购买

### 4.5 管理购物车

#### 添加商品到购物车
1. 在商品列表或详情页点击"加入购物车"
2. 系统提示添加成功
3. 购物车图标显示商品数量

#### 查看购物车
1. 点击导航栏"购物车"进入购物车页面
2. 查看所有已添加的商品

#### 操作购物车商品
- **全选/单选**：勾选需要结算的商品
- **修改数量**：点击"+"或"-"调整购买数量
- **删除商品**：点击"删除"按钮移除商品
- **清空购物车**：点击"清空"按钮删除所有商品

#### 计算总价
- 系统实时计算选中商品的总金额
- 显示商品总额、运费、订单总额
- 满50元免运费，否则收取8元运费

### 4.6 下单支付

#### 步骤1：进入订单确认
1. 在购物车中勾选要购买的商品
2. 点击"去结算"按钮
3. 进入订单确认页面

#### 步骤2：填写收货信息
- **收货人**：填写真实姓名
- **联系电话**：填写11位手机号
- **收货地址**：填写详细地址（省市区+街道门牌）

#### 步骤3：选择配送方式
- **快递配送**：送货上门（默认）
- **自提**：到指定地点自提

#### 步骤4：选择支付方式
- **余额支付**：从账户余额扣款（推荐）
  - 余额充足时自动扣款并分配配送员
  - 余额不足时提示充值
- **货到付款**：收到货物后现金支付

#### 步骤5：填写备注（可选）
- 可填写特殊要求，如：
  - "请放在门口"
  - "中午12点后配送"
  - "不要放辣椒"

#### 步骤6：提交订单
1. 确认订单信息无误
2. 点击"提交订单"按钮
3. 系统生成订单编号
4. 根据支付方式执行相应操作：
   - 余额支付：立即扣款，订单状态变为"配送中"
   - 货到付款：订单状态变为"待支付"

**订单状态说明：**
- **待支付（0）**：订单已创建，等待支付
- **配送中（1）**：已支付，配送员正在配送
- **待收货（2）**：配送员已送达，等待确认
- **已完成（3）**：用户已确认收货
- **已取消（4）**：订单已取消

### 4.7 查看和管理订单

#### 进入订单页面
点击导航栏"我的订单"

#### 按状态筛选
- **全部订单**：显示所有订单
- **待支付**：需要支付的订单
- **配送中**：正在配送的订单
- **待收货**：等待确认收货的订单
- **已完成**：已完成的订单
- **已取消**：已取消的订单

#### 订单操作

**待支付订单：**
- **立即支付**：进行余额支付
- **取消订单**：取消订单，恢复商品库存

**配送中订单：**
- **查看进度**：查看配送员信息和预计送达时间

**待收货订单：**
- **确认收货**：确认收到货物，订单状态变为"已完成"

**已完成订单：**
- **评价商品**：对购买的商品进行评分和评价
- **删除订单**：从列表中删除订单记录

**所有订单：**
- **查看详情**：查看订单完整信息
- **再次购买**：将订单商品重新加入购物车

### 4.8 个人中心

#### 查看个人信息
1. 点击导航栏"个人中心"
2. 显示以下信息：
   - 用户名
   - 手机号
   - 角色类型
   - 账户余额
   - 注册时间

#### 修改个人资料
1. 点击"修改资料"按钮
2. 可修改：
   - 用户名（不能与其他用户重复）
   - 手机号（不能与其他用户重复）
3. 点击"保存"完成修改

#### 修改密码
1. 点击"修改密码"按钮
2. 输入：
   - 旧密码
   - 新密码
   - 确认新密码
3. 点击"保存"完成修改

**密码要求：**
- 长度6-20个字符
- 建议包含字母和数字
- 定期更换密码保障安全

#### 账户充值
1. 点击"立刻充值"按钮
2. 弹出充值对话框
3. 选择充值金额或自定义输入
4. 点击"确认充值"
5. 充值成功后余额实时更新

**充值说明：**
- 最小充值金额：10元
- 最大充值金额：5000元
- 充值仅用于测试，实际项目需接入支付接口

### 4.9 评价商品

#### 提交评价
1. 在"已完成"订单中找到要评价的商品
2. 点击"评价"按钮
3. 填写评价信息：
   - **评分**：1-5星（5星为非常满意）
   - **评价内容**：文字描述使用体验（可选）
4. 点击"提交评价"

#### 查看评价历史
1. 点击导航栏"我的评价"
2. 查看所有历史评价记录
3. 显示：
   - 评价商品
   - 评分
   - 评价内容
   - 评价时间

**评价规则：**
- 只有"已完成"状态的订单才能评价
- 每个订单只能评价一次
- 评价提交后不可修改

### 4.10 退出登录

1. 点击导航栏右上角用户名
2. 选择"退出登录"
3. 系统清除登录状态
4. 跳转回登录页面

**安全提示：**
- 使用完毕后请及时退出登录
- 公共电脑使用后务必退出

## 五、管理员操作指南

### 5.1 管理员登录

1. 在登录页面输入管理员账号和密码
2. 选择角色：**管理员**
3. 点击"登录"按钮
4. 进入商品管理页面

**默认管理员账号：**
- 用户名：admin
- 密码：admin123

### 5.2 商品管理

#### 查看商品列表
1. 点击"商品管理"菜单
2. 显示所有商品（包括上架和下架）
3. 可查看：
   - 商品图片
   - 商品名称
   - 产地
   - 规格分类
   - 价格
   - 库存
   - 上架状态
   - 点击量

#### 搜索商品
1. 在搜索框输入关键词
2. 可按商品名称或产地搜索
3. 点击"搜索"按钮

#### 筛选商品
1. 选择规格分类：
   - 全部
   - 新鲜蔬菜
   - 有机蔬菜
   - 新鲜水果
2. 选择上架状态：
   - 全部
   - 上架
   - 下架

#### 新增商品
1. 点击"新增商品"按钮
2. 填写商品信息：
   - **商品名称**：必填
   - **产地**：必填，默认"未知"
   - **商品图片URL**：选填
   - **商品简介**：选填，默认"暂无简介"
   - **规格分类**：必选（新鲜蔬菜/有机蔬菜/新鲜水果）
   - **库存数量**：必填，默认0
   - **商品价格**：必填
   - **上架状态**：必选（上架/下架）
3. 点击"确定"保存

#### 编辑商品
1. 找到要编辑的商品
2. 点击"编辑"按钮
3. 修改商品信息
4. 点击"确定"保存

#### 删除商品
1. 找到要删除的商品
2. 点击"删除"按钮
3. 确认删除操作
4. 商品从数据库中永久删除

**注意：** 删除前请确认该商品没有被订单引用

#### 上下架操作
- **上架**：点击"上架"按钮，商品对用户可见
- **下架**：点击"下架"按钮，商品对用户隐藏

**应用场景：**
- 缺货时下架商品
- 季节性商品适时上下架
- 促销活动期间临时下架

### 5.3 订单管理

1. 点击"订单管理"菜单
2. 显示所有用户订单
3. 可查看：
   - 订单编号
   - 用户姓名
   - 收货信息
   - 商品清单
   - 订单金额
   - 支付方式
   - 订单状态
   - 配送员
   - 下单时间

#### 搜索订单
- **按用户姓名搜索**：输入用户名查找该用户的所有订单
- **按配送员姓名搜索**：输入配送员姓名查找其负责的订单

#### 筛选订单
按订单状态筛选：
- 全部
- 待支付
- 配送中
- 待收货
- 已完成
- 已取消

#### 查看订单详情
1. 点击订单行的"详情"按钮
2. 查看完整订单信息：
   - 用户信息
   - 收货信息
   - 商品明细（名称、单价、数量、小计）
   - 金额明细（商品总额、运费、订单总额）
   - 支付信息
   - 配送信息
   - 时间信息

**管理员权限：**
- 管理员可以查看所有订单
- 管理员不能修改订单状态（由系统自动流转）

### 5.4 用户管理

1. 点击"用户管理"菜单
2. 显示所有普通用户（role='user'）
3. 可查看：
   - 用户ID
   - 用户名
   - 手机号
   - 身份证号（如有）
   - 账户余额
   - 注册时间

#### 搜索用户
- **按用户名搜索**：模糊匹配用户名
- **按手机号搜索**：精确匹配手机号

#### 用户信息
- 管理员只能查看用户信息
- 管理员不能修改或删除用户账号
- 保护用户隐私和数据安全

### 5.5 配送员管理

#### 查看配送员列表
1. 点击"配送员管理"菜单
2. 显示所有配送员（role='PEISONG'）
3. 可查看：
   - 配送员ID
   - 用户名
   - 手机号
   - 身份证号（如有）
   - 当前状态（空闲/配送中）
   - 账户余额

#### 搜索配送员
- **按用户名搜索**：模糊匹配用户名
- **按手机号搜索**：精确匹配手机号

#### 配送员状态
- **空闲**：可以接收新订单
- **配送中**：正在配送订单，暂不分配新订单

**说明：**
- 配送员状态由系统自动更新
- 支付成功后自动分配空闲配送员
- 配送员确认送达后状态恢复为"空闲"

## 六、配送员操作指南

### 6.1 配送员登录

2. 选择角色：**配送员**
3. 点击"登录"按钮
4. 进入配送订单页面

**测试配送员账号示例：**
- 用户名：陈紫凡 / 李伟健 / 韩梅梅 等
- 密码：123456（大部分配送员）
- 陈紫凡密码：55524211

### 6.2 查看配送订单

#### 订单分类
- **全部订单**：显示该配送员负责的所有订单
- **配送中**：正在配送的订单
- **已送达**：已完成配送的订单

#### 订单信息
每个订单显示：
- 订单编号
- 收货人姓名
- 联系电话
- 收货地址
- 商品清单（名称、数量、规格）
- 订单总额
- 支付方式
- 下单时间
- 备注信息

### 6.3 确认送达

#### 操作步骤
1. 在"配送中"标签下找到对应订单
2. 核对收货信息
3. 联系收货人确认送达
4. 点击"确认送达"按钮
5. 系统更新订单状态为"已送达"
6. 配送员状态恢复为"空闲"

#### 注意事项
- 确认送达前务必核实收货人身份
- 货到付款订单需收取现金
- 如遇问题及时联系管理员

### 6.4 查看个人信息

1. 点击导航栏"个人中心"
2. 查看个人信息：
   - 用户名
   - 手机号
   - 角色：PEISONG
   - 当前状态
   - 账户余额

3. 可修改资料和密码（同普通用户）

## 七、常见问题

### 7.1 账号相关问题

**Q1：忘记密码怎么办？**
A：目前系统不支持自助找回密码，请联系管理员重置密码。

**Q2：用户名或手机号已被注册？**
A：用户名和手机号必须唯一，请使用其他用户名或手机号注册。

**Q3：如何修改绑定的手机号？**
A：登录后进入"个人中心"，点击"修改资料"，输入新的手机号（不能被其他用户使用）。

**Q4：登录时提示"令牌无效"？**
A：登录状态已过期（24小时），请重新登录。

### 7.2 购物相关问题

**Q5：为什么有些商品看不到？**
A：只显示"上架"状态的商品，下架商品对用户不可见。

**Q6：购物车商品失效了？**
A：可能原因：
- 商品已下架
- 商品库存不足
- 商品被删除
请重新添加可用商品。

**Q7：如何修改购物车商品数量？**
A：在购物车页面，点击商品数量旁的"+"或"-"按钮调整。

**Q8：为什么不能结算？**
A：检查以下项：
- 是否勾选了至少一件商品
- 商品是否有足够库存
- 商品是否仍在上架状态

### 7.3 订单相关问题

**Q9：下单后能修改订单吗？**
A：订单一旦创建不能修改，如需更改请取消订单后重新下单。

**Q10：如何取消订单？**
A：在"待支付"状态的订单中，点击"取消订单"按钮。

**Q11：取消订单后库存会恢复吗？**
A：是的，取消订单后商品库存会自动恢复。

**Q12：余额支付失败怎么办？**
A：检查账户余额是否充足，如不足请先充值或选择"货到付款"。

**Q13：订单状态长时间不更新？**
A：可能原因：
- 没有空闲配送员
- 系统异常
请联系管理员处理。

**Q14：能同时下多个订单吗？**
A：可以，系统支持同时存在多个订单。

### 7.4 支付相关问题

**Q15：支持哪些支付方式？**
A：目前支持两种支付方式：
- 余额支付：从账户余额扣款
- 货到付款：收到货物后现金支付

**Q16：如何充值余额？**
A：进入"个人中心"，点击"立刻充值"，选择或输入充值金额。

**Q17：余额支付的优点是什么？**
A：
- 支付成功后立即分配配送员
- 订单状态自动变为"配送中"
- 无需等待确认支付

**Q18：货到付款需要注意什么？**
A：
- 准备好现金
- 收货时当场付款
- 订单状态需管理员手动更新

### 7.5 配送相关问题

**Q19：多久能送达？**
A：配送时间取决于配送员安排，通常1-2小时内送达。

**Q20：能指定配送时间吗？**
A：可在订单备注中说明期望的配送时间，但不能保证准时。

**Q21：如何查看配送进度？**
A：在"配送中"订单中点击"查看进度"，可看到配送员信息。

**Q22：配送员联系不上怎么办？**
A：请联系客服或管理员协调处理。

### 7.6 评价相关问题

**Q23：什么时候可以评价？**
A：订单状态为"已完成"时可以评价。

**Q24：评价后可以修改吗？**
A：评价提交后不能修改，请谨慎填写。

**Q25：不评价会影响下次购物吗？**
A：不会，评价是自愿的，不影响购物功能。
